# Threads 账号合规

本文件处理账号层风险，不只看单篇文案，也看账号资料、行为模式、申诉路径。

证据使用原则：
- 账号状态、停用、仿冒、版权与资料页修改，优先使用官方帮助中心页面。
- 若把 Instagram 的推荐资格规则外推到 Threads 增长表现，而页面没有直接写 Threads，必须标记为 `Inference`。
- 账号资料页审核完成后，回到 `threads-review-workflow.md` 汇总最终结论。

## 一、账号层常见风险

### 1. 身份真实性风险
- 冒充真实人物、机构、媒体或专家
- 名称、头像、Bio、链接让用户误以为是官方或权威认证
- 权威头衔写得过满、无根据、无法自证

### 2. 推荐资格风险
- 头像、Bio、链接或公开内容触发 Recommendations Guidelines
- 账号处于不适合推荐状态，导致内容不易被陌生用户看见

### 3. 行为模式风险
- 异常频繁重复发布
- 重复评论、机械互动、刷量嫌疑
- 未经同意反复私信商业内容
- 诱导用户用评论、关注、抽奖换利益

### 4. 安全与产权风险
- 被仿冒
- 使用他人图像、截图、文章、图表无授权
- 遭遇钓鱼、假客服、假申诉

## 二、账号资料合规清单

### 头像
- 不使用可能造成误认的机构 Logo，除非确有授权
- 金融账号不建议用“官方风”标识冒充监管、券商、基金或媒体

### 名称
- 真实人设可以使用真实姓名
- 若不是官方机构，不要在账号名中暗示“官方”“唯一授权”“认证中心”等

### Bio
- 可写专业背景，但不要夸大、杜撰、伪造认证
- 不要写承诺收益、稳赚、带单、翻倍、保本
- 不要用暗示违规交易或灰产服务的表述

### Link
- 不要导向可疑跳转页、收集敏感信息页、诈骗页
- 不要导向违规金融拉人头、非法代操、未经许可的高风险产品销售页

## 三、金融账号特别注意

### 高风险账号特征
- 自称内幕、老师、保赚、翻仓
- 高收益截图、资金曲线却无可核实来源
- 不断把用户往私域导流成交
- 用“免费诊股”“私发代码”“留言 666 拉群”做强互动诱导

### 相对稳妥的账号特征
- 教育型、分析型、框架型表达
- 有真实背景但不过度神化履历
- 强调风险、条件、情境而非绝对结论
- 引导公开讨论多于私信成交

## 四、违规与处理

### 如果账号被限流或不适合推荐
1. 先查 Account Status
2. 看是内容、头像、Bio 还是链接触发问题
3. 删除或修改问题项
4. 若判断错误，可按官方入口申请 review

### 如果 Threads profile 被停用
- 官方说明：如果用来创建 Threads 的 Instagram 或 Facebook 账号被停用，Threads profile 也会被停用
- 先查基础账号状态，再查 Threads profile 本身

### 如果被仿冒
- 用官方 impersonation 举报入口处理
- 如果你是代表他人举报，准备好必要身份和授权资料

### 如果涉及版权/商标
- 走 Threads/Instagram 的 IP 举报或撤回流程
- 不建议在公开帖文里点名互撕，容易演变成骚扰
