# P2 每日早评早报

## 目标
让用户在开盘前快速看懂：昨天到今天发生了什么、今天台股可能怎么走、该盯哪些板块和股票。

## 固定结构
1. 隔夜全球市场重点
2. 今日影响台湾市场的关键新闻
3. 对大盘风格的判断
4. 今日重点板块
5. 板块龙头与热点股
6. 重点股深度分析
7. 交易参考与风险提醒

## 必看变量
- 美股三大指数、费半、NVIDIA/台积电 ADR
- 美债收益率、美元指数、油金走势
- 联准会预期、地缘政治
- 台币汇率、台指期、三大法人、政策与产业消息

## 输出要求
- 明确区分：宏观新闻、产业新闻、公司新闻
- 只保留会影响资金流向的新闻
- 每个重点板块要回答“为什么今天轮到它”
- 每档重点股都要附：基本面一句话、技术面一句话、量价一句话、价位一句话
- 必须承接前一天盘后复盘，说明哪些判断延续、哪些变量新增

## 价位表达规范
- 观察区间：适合先观察或轻仓试单的位置
- 分批布局区间：逻辑成立但不宜一次性重仓
- 突破确认价：站上后强势结构较完整
- 失效位：跌破后原判断暂缓
