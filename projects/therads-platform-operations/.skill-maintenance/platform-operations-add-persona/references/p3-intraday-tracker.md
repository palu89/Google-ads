# P3 盘中跟踪

## 目标
验证早评逻辑有没有兑现，并根据盘中新消息和趋势变化及时修正判断。

## 盘中优先检查
1. 大盘环境是否维持
2. 热点板块是否继续发酵
3. 龙头股是否还领涨
4. 盘中是否有新增新闻、公告、法人动态
5. 个股量价是否支持原判断

## 单篇结构
1. 早上为什么看这档
2. 盘中新增了什么
3. 板块今天是增强、分化还是退潮
4. 个股现在偏多、转震荡还是转弱
5. 关键价位有没有突破或跌破
6. 接下来该观察、等确认、等回踩还是暂缓

## 输出提醒
- 盘中内容重在“现在和早上相比，哪里变了”。
- 不要把早评原文重写一遍。
- 多用动态词：站稳、失守、放量、缩量、承接、转弱、量缩震荡。
- 必须回扣早评中的原判断，明确写出是验证、加强、还是修正
