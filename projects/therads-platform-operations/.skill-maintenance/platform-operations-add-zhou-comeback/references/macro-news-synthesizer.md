# 宏观与新闻归纳方法

## 新闻分层
1. 全球宏观：利率、通胀、美元、美债、能源、地缘政治、美股与费半
2. 台湾宏观：汇率、出口、PMI、CPI、央行、政策、期货与 ADR
3. 产业新闻：AI、半导体、伺服器、航运、金融、生技等
4. 公司新闻：财报、法说、订单、资本支出、并购、处分、除息

## 处理顺序
1. 抓出时间最接近市场的最新事件
2. 明确事件发生日期与来源
3. 判断它影响的是风险偏好、产业预期，还是单一公司
4. 映射到台股：大盘风格 -> 板块 -> 龙头 -> 个股

## 过滤原则
- 新闻多不代表重要，先找能改变资金流向的事件
- 同一条新闻若已被市场消化，要写“延续效应”而不是重新当成新驱动
- 对实时内容必须核实时间，避免把旧新闻当成当天催化

## 输出句式
- 先写“事件是什么”
- 再写“为什么影响台股”
- 最后写“影响到哪个板块、哪类股票”
