**创建日期**: 2026-04-08

**适用对象**: 与JP11.1同类的小型「投資助言·代理業」LINE导流型广告主  
**数据可信度说明**: 本文件中的公司名为AI推断，未经金融厅官网验证，仅保留行为模式分析

## 一、小型竞品三大模式分类

### 1.1 AI诊断诱饵系（The Diagnosis Bait）
- **LP特征**: 极简800-1200px，深色背景，绿色LINE按钮
- **首屏**: "30秒でAI株診断" + AI芯片插画 + LINE CTA
- **转化路径**: 点击CTA → LINE加粉 → 机器人回复 → 人工+机器人混合 → VIP群邀请
- **牌照位置**: Footer最底部

**典型文案**:
- 标题: `AIが導き出す「急騰予測銘柄」` / `あなたの株をAIが診断` / `無料！AI銘柄分析レポート配信`
- 描述: `プロ投資家も利用するAI解析。LINE追加で診断結果を無料公開中。今すぐチェック。`

### 1.2 LINE即时信号系（The Messenger Notification）
- **LP特征**: 约1200px，含历史胜率图表
- **首屏**: "毎日の推奨銘柄配信" + 手机聊天截图 + 友だち追加
- **转化路径**: LP → LINE → 每日信号推送 → 付费VIP
- **牌照位置**: Header页眉悬浮（比诊断系合规意识强）

**典型文案**:
- 标题: `LINEで届く推奨銘柄シグナル` / `株の買い時はLINEで配信`
- 描述: `「明日上がる株」を厳選配信。忙しい方にもおすすめ。友だち追加で最新情報を受け取る。`

### 1.3 免费咨询/助言系（The Consultation Hook）
- **LP特征**: 内容型，含专家头像和FAQ
- **首屏**: "投資家限定コミュニティ" + 专家头像 + LINE追加
- **转化路径**: LP → LINE → 免费咨询 → 付费助言契约
- **牌照位置**: Footer底部

**典型文案**:
- 标题: `投資顧問による個別銘柄相談` / `投資助言のプロにLINEで相談`
- 描述: `金融庁登録の投資顧問がサポート。今の株価で売るべきか待つべきか、無料で診断します。`

## 二、日本用户防御心理（5ch/Twitter口碑清洗）

### 2.1 用户眼中的LP特征
用户在讨论区的典型描述：
- **"极简黑/深蓝"**: "又是那种一看就很像投資詐欺的黒底蛍光绿按钮，但下面真的有公司信息和金商牌照"
- **"AI诊断组件"**: 不论输入什么股票代码，最后都跳出"诊断完成，点击LINE获取详细分析"
- **倒计时功能**: "限时免费诊断"/"今日仅剩名额"制造紧迫感

### 2.2 用户的三道防线
1. **"牌照检查员"**: 先点LP底部链接查金商番号，查不到直接拉黑
2. **"关键词搜索测试"**: 公司名+「評判」或「詐欺」放Google搜一遍
3. **"LINE行为鉴别"**: 加LINE后被强制拉群 = 杀猪盘，立刻举报

### 2.3 背景：LINE詐欺 泛滥
- 近年LINE投资诈骗极其泛滥，已产生专用术语「LINE詐欺」
- 日本投资者对"投资建议类LINE导流"保持极高怀疑度
- 持有金商番号是唯一能让用户从"诈骗"判定转为"激进但合法"的因素

## 三、投放策略逆向推断矩阵

| 维度 | 策略推断 | 可信度 |
| :--- | :--- | :--- |
| **关键词** | 高意图长尾词 + 竞品拦截（如`[个股名] 分析`、`投資顧問 評判`） | [推断] |
| **出价** | 手动CPC / eCPC + 受众调整，50-100有效转化后切tCPA | [行业常识] |
| **时段** | 08:30-10:00（开盘）+ 17:00-21:00（复盘），不全天投 | [推断] |
| **地域** | 东京/大阪/名古屋大区 + 总部所在地强化 | [推断] |
| **月预算** | 30万-80万JPY（日均10-20条线索） | [估计] |
| **渠道** | Search 80% + 精准Retargeting 20%，极少投YouTube（制作成本高） | [推断] |

### 3.1 关键词策略详解
- **绝不买泛词**（"开户"等 → KYC审核复杂，漏斗太深）
- **买"诊断需求"和"策略需求"**: `配当金 少ない`、`銘柄診断 LINE`、`投資顧問 評判`
- **竞品拦截**: 买"SBI荐股工具怎么样"类词，截获已有开户意愿但对选股没信心的用户

### 3.2 出价策略详解
- Smart Bidding（tROAS）在样本不足时表现极糟
- 初期用手动CPC控制点击质量
- 系统积累足够"有效进LINE群"数据后再切自动

### 3.3 时段与地域详解
- 投放时段极度依赖客服在线时间
- 如果是顾问本人维护LINE → 无人时段关停广告
- 地域定向不仅为转化，更为**物理信任**（素材中强调"福岡本社"可大幅提升当地点击意图）

### 3.4 素材策略详解
- **AI诊断系**: 极度偏好Search Ads + 移动端LP（PC端需扫码加LINE = 浪费）
- **信号通知系**: 占据YouTube广告（视频展示账面收益"跳动"激发FOMO）
- **GDN素材**: "报表化"风格 — 深色底、巨大收益百分比、专家头像

## 五、对JP11.1的竞争策略建议

### 5.1 差异化定位（避开同行路径依赖）
- 大家都在买「AI株式診断 LINE」→ 词贵且被用户标记为"骚扰"
- **策略**: 强调"基于金商牌照的合规诊疗"，拼**"専門合規の個股体検"**而非拼AI

### 5.2 缓解"被欺诈恐惧"（Anti-Scam Shield）
LP必须加入针对性声明：
> 「弊社は金融庁認可の投資助言代理業者です。LINE追加後に強制的なグループ入会は一切ございません。個別の投資アドバイスの提供を目的としており、安心してお使いいただけます。」

### 5.3 提升AI诊断可信度
- 用户不信任"AI"（认为只是营销手段）
- **优化**: AI诊断过程可视化（分析股价波動率→計算移動平均線→生成財務健康評分）
- 评分越有"学术感/报表感"，被视为营销手段的可能性越低

### 5.4 重塑LINE咨询流程
- 日本股民极讨厌无意义的"人工寒暄"
- **优化**: LINE自动欢迎词中**直接给结论**（诊断结论+下一步建议）
- 把诊断报告（PDF/网页）直接回传，作为服务交换价值

### 5.5 高危词规避
- 竞品在用「急騰（暴涨）」等词 → Google Misrepresentation模型高危词
- **替代用词**: 「分析」「適正株価（目标股价）」「資産形成」
- 既能引流又获得Quality Score青睐，降低CPC

### 5.6 广告描述中前置牌照号
已验证竞品中信任度最高的做法：在广告描述中直接写入金商番号
- 示例: `【福岡財務支局長 第69号】専門家による株式診断。LINE追加で診断レポートを無料提供。`

### 5.7 PMax使用注意
- 素材中不要过度使用"LINE"等社交倾向语
- 否则PMax会把素材分发到低质社交版块
- 强导流文案限制在Search Ads中（搜索意图最纯粹）

### 5.8 数据层降维打击
- 竞品在用手动CPC摸索 → 你通过高强度OCT回传实现算法优势
- 系统学习到"谁是真正愿意付费讨论投资"后，精准覆盖速度远超手动调价竞品

### 5.9 预算切分建议
- 80%预算 → Search Ads，拦截搜索"SBI评级"/"乐天个股数据"的用户
- 这些用户正处于"决策瘫痪"，是AI诊断的最佳切入点

## 六、竞品LP审计框架（5-Tier模型）

当获取真实竞品URL后，按以下层级执行审计：

### Tier 1: 身份合法性（Misrepresentation Check）
- 牌照号位置：`display:none`或footer最底部 → 大幅降低Quality Score
- 验证标准：FSA要求"清晰且立即可见"

### Tier 2: 页面独立价值（Destination Compliance）
- "AI诊断+LINE加粉"模式：除按钮外无AI运作说明或公司信息 = Thin Bridge Page
- 计算Section信息密度，800px极简页极易违规

### Tier 3: 信任溢价（Trust Premium）
- 检查：社交认同（使用人数）/ 专业认同（专家头像）/ 技术认同（诊断模型示意图）
- 判断竞品用什么视觉要素消除用户恐惧

### Tier 4: CTA摩擦力（Friction Audit）
- LINE按钮前是否有"铺垫性"文案
- 铺垫能区分"随便点点"的泛流量和"有诊断需求"的真股民

### Tier 5: 法律防线（Compliance/Liability）
- 免责条款：必须有投资损失提示
- 无`過去の実績は将来の成果を保証するものではありません`→ 法律不合格
