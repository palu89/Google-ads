# 日本金融投资类Google Ads合规监管完整指南

**更新日期**: 2026-04-08  
**重要性**: ⭐⭐⭐ 强制要求  
**适用范围**: 所有日本金融投资类Google Ads投放

---

## 摘要

日本的金融投资类广告需要遵守**Google官方政策**、**日本金融庁（FSA）**、**日本証券業協会（JSDA）**的三重严格监管。不合规的广告将导致拒审、账户被禁或法律风险。

---

## 1. Google Ads 日本金融广告政策限制

### 1.1 广告主验证要求（在投放前必须完成）

#### **第一层：广告主身份验证**
- 适用范围：所有敏感类别广告主
- 验证内容：确认商家、服务及产品的合法性和真实性
- 审核周期：通常 1-2 周

#### **第二层：金融服务专项验证**（关键）
- **强制要求**：投放金融投资类广告前，必须完成 **Financial Services Verification**
- **证明材料**：必须证明拥有日本金融庁（FSA）颁发的相应业务牌照
- **适用产品类型**：
  - 证券交易（株式、投资信托）
  - NISA / iDeCo
  - 外汇（FX）和差价合约（CFD）
  - 加密货币交易（仅限金融庁注册的平台）

**必需文件清单**：
```
□ 金融商品取引業登録証書（金商牌照副本）
□ 業務基本方針書（营业方针书）
□ 会社登記簿謄本（法人登记簿）
□ 代表者身分証明書（代表人身份证明）
□ Google Ads账户所有者身份证明
```

### 1.2 落地页强制要求

投放前，落地页必须清晰展示以下内容：

| 必需项 | 具体要求 | 位置 |
|:---|:---|:---|
| **业务地址** | 完整的日本国内地址，不能仅为邮箱 | 页面底部（Footer） |
| **联系电话** | 有效的日本本地电话号码 | 页面底部 |
| **手续费/费用结构** | 清晰列出所有可能的费用（手数料、点差、管理费等） | 易于查找的位置（通常为"費用"或"手数料"页面） |
| **金融庁登録番号** | 「金融商品取引業者 関東財務局長（金商）第XXXX号」 | 页面底部，必须防止用户误点 |
| **加入协会信息** | 日本証券業協会等协会成员身份 | 页面底部 |

| 产品 | 政策 | 说明 |
|:---|:---|:---|

| **期权交易** | ✅ 允许 | 需要完整的风险说明 |

## 2. 禁止词汇与被拒表述

### 2.1 Google Ads 平台禁止词汇

| 词汇类别 | 禁止词汇（日语） | 中文翻译 | 拒审原因 |
|:---|:---|:---|:---|
| **收益承诺** | 絶対儲かる、必ず勝てる、利益保証 | 绝对赚钱 / 必胜 / 保证收益 | 禁止承诺特定投资结果 |
| **短期暴富** | 一発逆転、簡単に稼げる、誰でも稼げる、即日100万 | 一夜翻身 / 轻松赚钱 / 人人都能赚 / 当天赚100万 | 夸大宣传、未经证实 |
| **无风险宣传** | リスクゼロ、損をしない、100%安全、寝ていても稼げる | 零风险 / 不会亏 / 100%安全 / 睡觉也赚钱 | 违反风险披露标准 |
| **金融捷径** | 即時承認、即金、確実、間違いなく | 即时审批 / 即时到账 / 绝对 / 肯定会 | Google全球禁止 |
| **未验证超级** | No.1（无数据支撑）、最高、最大級| No.1 / 最好 / 最大级 | 需实质性证据支持 |
| **极端夸大** | 爆益、暴利 | 爆炸性收益 | 触发不当表示禁令 |

### 2.2 日本法律（金融商品取引法）禁止词汇

**禁止原因**：《金融商品取引法》规定："断定的判断（断定的判断の提供）是非法的"

| 禁止场景 | 违规表述 | 合规改写 |
|:---|:---|:---|
| **对未来市场做出确定表述** | 「値上がりする」「確実に上昇する」 | 「上昇する可能性があります」「歴史的には上昇傾向です」 |
| **隐瞒风险** | 「初心者でも確実に利益」「元本保証」 | 「初心者向けの情報提供」「元本割れの可能性があります」 |
| **对比性夸大** | 「業界最低の手数料」「最も安全」（无证据） | 「A社と比較して手数料が低い」（附上具体对比） |
| **时间承诺过度** | 「1ヶ月で倍になる」「3日以内に利益」 | 「長期的な資産形成をサポート」 |

**日本《景品表示法》补充规定**：
- 禁止使用虚假或误导性陈述
- 广告中利益用大字、限定词用小字 = 违规
- 所有表述都必须基于事实和数据

## 3. 强制免责声明（Disclaimer）& 字体要求

### 3.1 日本金融厅对字体的严格要求

> **核心原则**："对于风险的说明必须要在整个广告中用最大的字体，必须与其他内容明显不同。"

| 要求项 | 具体规定 |
|:---|:---|
| **风险提示字体** | 必须是整个广告中的**最大字体** |
| **视觉区分度** | 必须与其他内容**明显不同**（不能淡灰色、不能用背景淡化） |
| **禁止做法** | ❌ 将风险提示用极小字体或颜色淡化呈现 |
| **违规案例** | ❌ "手数料最低从50円开始" （数字大，限定词小） |

#### **通用字体基准**：
- **网页落地页**：风险提示不小于 **8pt-10pt**（或13级以上）
- **移动端**：必须无需缩放即可清晰辨认
- **视频广告**：风险提示停留至少 **2-3秒**，字体清晰可读
- **对比度**：与背景色彩对比度不低于 **4.5:1**（WCAG标准）

### 3.2 通用免责声明文案

#### **投资风险基础声明**（必须）
```
【ご注意事項】
弊社の取扱商品は、価格変動等により投資元本を割り込むおそれがあります。
ご投資にあたっては、契約締結前交付書面や目論見書をよくお読みください。
```

#### **业绩声明**（涉及历史数据时必须）
```
「過去の実績であり将来を約束するものではありません」
（仅为过往业绩，并不保证未来收益）
```

#### **税务声明**（涉及税务或NISA/iDeCo时）
```
「税制は変更される可能性があります。詳細は税理士等の専門家にご確認ください」
（税制可能发生变更，详情请咨询税务师等专业人士）
```

#### **手续费声明**（必需，完整披露）
```
「別途手数料がかかる場合があります。詳細は当社ウェブサイトをご確認ください」
（可能另需手续费，详情请参阅本公司网站）
```

### 3.3 风险提示文字占比要求

根据 Google Ads 和日本金融厅标准：
- **最小占比**：广告面积的 **15%** 以上
- **位置**：必须在用户初次看到时能看到，不能滚动后才出现

## 4. 金融庁登録番号的展示要求

### 4.1 是否必须展示？

**是的，必须展示。** 这是合法券商的"身份证"，也是广告通过Google审核的先决条件。

### 4.2 展示内容与格式

**完整格式**：
```
商号等：[您的券商名称] 証券株式会社
金融商品取引業者 関東財務局長（金商）第[XXXX]号
加入協会：日本証券業協会、一般社団法人金融先物取引業協会
```

**其他财务局示例**：
- 関東財務局（Kanto）= 东京、神奈川等地区
- 近畿財務局（Kansai）= 大阪、京都等地区
- 中部財務局（Chubu）= 名古屋地区
- etc.

### 4.3 展示位置

| 场景 | 推荐位置 | 要求 |
|:---|:---|:---|
| **网站/LP底部** | 页脚（Footer）区域 | 必须在**所有页面**都显示 |
| **广告素材** | 视频广告结尾或角落 | 停留2-3秒，清晰可读 |
| **官方网站首页** | 仅底部或"会社概要"页面 | 最少在一处显示 |
| **"关于我们"页面** | 专门的"会社概要"或"法人信息"区域 | 完整注册信息 |

**最佳实践示例**：
```html
<!-- Footer区域 -->
<footer>
  <div class="compliance-info">
    <p>金融商品取引業者 関東財務局長（金商）第[XXXX]号</p>
    <p>加入協会：日本証券業協会</p>
    <p>〒100-0005 東京都千代田区丸の内[X-X-X] [会社名]ビル</p>
  </div>
</footer>
```

## 5. NISA / iDeCo 广告的特殊合规要求

### 5.1 适用的法律框架

- **金融商品取引法（FIEA）**：证券投资类广告基础法规
- **景品表示法（不当景品類及不当表示防止法）**：禁止不当表示
- **金融厅广告监管标准**：由日本金融厅（FSA）制定

### 5.2 NISA（少额投资非课税制度）广告特殊要求

#### **必须说明的制度限制**

| 限制项 | 必需说明 | 示例文案 |
|:---|:---|:---|
| **损益流转限制** | 必须标注亏损不能抵扣普通账户收益 | 「NISA口座内の損失は、他の口座の利益と損益通算できません」 |
| **免税额度上限** | つみたて投資枠 360万円 + 成長投資枠 240万円 | 「本年度の非課税投資枠は合計600万円です」 |
| **复投资要求** | NISA账户需要持续投资（不能仅持有） | 「継続的な投資をお勧めします」 |
| **制度有效期** | 明确提及制度时间限制（如新NISA的持续期限） | 「新NISAは2024年から開始された無期限の免税制度です」 |

#### **禁止在NISA广告中做的事**
- ❌ 推荐具体的股票或基金（可能触发投资顾问业注册义务）
- ❌ 暗示NISA账户可以"致富"或"高回报"
- ❌ 隐瞒"投资仍然存在亏损风险"
- ❌ 夸大"免税效果"的经济效益

#### **2024年新NISA关键更新需标注**
```
つみたて投資枠（积累投资枠）：最多 360万円
  - 年度投资限额：180万円
  - 投资期间：无期限
  
成長投資枠（成长投资枠）：最多 240万円
  - 年度投资限额：120万円
  - 可投资产品：高增长潜力股票及基金
```

### 5.3 iDeCo（个人型确定缴费养老金）广告特殊要求

#### **必须说明的限制条件**

| 限制项 | 必需说明 | 示例文案 |
|:---|:---|:---|
| **提取限制** | 原则上60岁前**无法提取** | 「原則60歳まで受給できません」 |
| **年龄限制** | 加入资格通常为20-60岁 | 「加入対象：20歳以上65歳未満の国民年金・厚生年金保険の加入者」 |
| **管理费** | 必须说明加入时及每月会产生管理费 | 「加入時および毎月の運営管理手数料がかかります」 |
| **受益人规定** | iDeCo为个人养老金，不能转移 | 「受取人の変更はできません」 |

#### **禁止在iDeCo广告中做的事**
- ❌ 与存款进行直接比较（如"比存款划算"）
- ❌ 对节税效果做出精确预测或承诺
- ❌ 向不符合年龄条件的人群投放
- ❌ 暗示可以提前提取

#### **涉及税务优惠时的合规声明**
```
「掛金が全額所得控除となり、受取時の税制優遇もあります
ただし、税制は変更される可能性があります。詳細は税理士等の専門家にご確認ください」
```

| **13. NISA合规** | NISA广告是否避免了对特定产品的推荐？ | ☐ | 仅提供信息 |
| **14. iDeCo合规** | iDeCo广告是否明确说明60岁前无法提取？ | ☐ | 必须标注 |
| **15. 记录保留** | 是否保留了所有广告的法律审查记录以备金融厅监察？ | ☐ | 需要存档备查 |

## 8. 金融厅审查时的需要文件与流程

### 8.1 需要长期保留的文件（备金融厅检查）

```
【法律合规文件夹】
├─ 广告审查记录（日期、审查人、批准人）
├─ 所有投放过的广告素材（图片、视频、文案）
├─ Landing Page 的每个版本快照
├─ 投诉记录及处理回复
├─ 金融庁的行政指导或处罚记录（若有）
├─ 内部的法律审查意见书
└─ 用户反馈与投诉处理记录
```

### 8.2 金融厅検査対応流程

1. **金融厅通知到达**（通常邮件或电话）
   - 确认检查原因（通常是投诉或例行监查）
   - 记录检查日期和检查官信息

2. **文件收集**（通常要求3-5个工作日内提交）
   - 汇总上述所有文件
   - 准备中文翻译版本（如有外文内容）
   - 按金融厅要求的格式整理

3. **现场检查或书面审查**
   - 金融厅可能要求上门检查或邮件提交
   - 配合审查官的质询

4. **整改**（若有发现）
   - 立即停止违规广告投放
   - 修改不合规内容
   - 提交书面整改报告

5. **验证检查**
   - 金融厅再次验证整改内容

## 10. 常见被拒原因与解决方案

### 被Google拒审

| 拒审原因 | 具体表述 | 解决方案 |
|:---|:---|:---|
| **Policy violation: Financial Services** | 广告被拒，理由为"违反金融服务政策" | 检查是否完成Financial Services Verification；检查是否有禁止词汇 |
| **Misleading claims** | 误导性声明 | 移除所有承诺性词汇（必胜、绝对、保证等）；添加风险声明 |
| **Landing page issues** | 落地页问题 | 添加清晰的金商号、办公地址、电话；修复页面加载速度 |
| **Unverified data** | 未验证的数据/图表 | 为所有性能数据、排名声明附上来源链接或说明 |

### 被日本金融厅监管部门关注

| 风险信号 | 触发原因 | 应对措施 |
|:---|:---|:---|
| **投诉举报** | 用户向金融厅投诉您的广告 | 立即停止使用被投诉的表述；改进产品或服务 |
| **例行监查** | 金融厅定期抽查 | 确保所有广告都符合最新指南 |
| **竞争对手举报** | 同行竞争举报 | 保留完整的广告记录和修改历史 |
| **舆论关注** | 社交媒体曝光或新闻报道 | 立即反应；与金融厅主动沟通 |
