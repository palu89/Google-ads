# 项目学习链

## 2026-03-21

- 空线程不适合直接写剧情，先搭控制平面更稳。
- 本项目必须把 `资料 -> 边界 -> 变量 -> 时间线 -> 人物 -> 卷章 -> 章节` 串成固定主线。
- `1927-1953` 的中国近代史项目，最危险的失真不是“记错一件事”，而是“跳过历史传播机制”。
- 以后每次输出都要明确区分：
  - `[硬史实]`
  - `[高可信推断]`
  - `[低可信推断]`
  - `[架空改写]`
- 主变量优先考虑 `组织 / 情报 / 后勤`，不要优先考虑高层操盘。
- 样板卷优先锁 `1927-1931`，再向前后扩展。

## 2026-03-24

- 用户已明确：王岩“必须是天选之人”“必须亲近教员”“前世就是教员接班人但接班失败”，这些属于本项目硬设，不能再被我擅自降格处理。
- 后续优化时，不能把“高规格主角”误判成需要削掉的风险项；真正该优化的是这些高规格设定在历史现场中的落地方式、传播链和代价。
- 当用户强调主角是工农兵底色、童工底色、实干家底色时，不能再把风险表述成“像是在讲后期语言”或“只会说不会做”；这种判断会直接偏离项目意图。
