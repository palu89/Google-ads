# 变量传播模型

## 六阶段硬规则

1. `注入`
   - 主角或变量首次进入历史现场。
   - 影响范围应控制在 `1-10` 人、`1` 个点位或 `1` 次任务。
2. `扎根`
   - 变量不再是孤立事件，而是形成稳定信任、稳定收益或稳定依赖。
   - 至少出现 `2` 次以上复用。
3. `扩散`
   - 变量跨越原始受益者，进入新的地理点位、组织链或社会关系。
   - 传播需要载体，例如干部、交通员、家族、商路、伤员、文件。
4. `对抗`
   - 变量开始遭遇既有制度、路线斗争、敌方侦察、资源不足或内部怀疑。
   - 这一阶段必须产生代价，不能只收收益。
5. `放大`
   - 变量开始改变更大范围的效率、节奏或人员存活率。
   - 典型表现是“让某项历史趋势加速、延后或转向”，而不是突然反转。
6. `改写`
   - 只有在前五阶段连续成立后，才允许进入全国级、跨战区级或跨年代级的历史偏移。

## 变量类型池

- `信息型`：预警、甄别、路线判断、错误规避。
- `组织型`：干部保全、联络纪律、建制磨合、执行效率。
- `后勤型`：粮秣、药品、转运、避险、隐蔽与交通。
- `生存型`：关键个体不死、不中断、不失联。
- `认知型`：有限度地纠正错误判断，但不能直接灌输未来答案。

## 当前推荐主变量

- `[高可信推断]` 推荐优先考虑 `组织 + 情报 + 后勤` 的复合型变量。
- 原因：这类变量容易局部起效、能长期复用、也最符合“渐进传播”。

## 变量放大判据

- 是否有稳定传播载体。
- 是否跨过至少一个组织边界。
- 是否引发可见的反制。
- 是否出现新增成本。
- 是否能在 `1 个月 / 1 年 / 10 年` 三个尺度上讲清后果。

## 反失控规则

- 没有代价，就不允许判定变量成立。
- 没有载体，就不允许判定变量扩散。
- 没有反制，就不允许判定变量进入放大。
- 没有完整链条，就不允许改写全国结局。

## 变量台账模板

| 字段 | 含义 |
| --- | --- |
| 变量编号 | 用于追踪同一条变量链 |
| 注入时间 | 第一次发生的绝对日期 |
| 注入地点 | 县 / 区 / 城市 / 战区 |
| 初始受影响者 | 第一批受益人或受损人 |
| 当前阶段 | 注入 / 扎根 / 扩散 / 对抗 / 放大 / 改写 |
| 传播载体 | 人、组织、交通线、制度缝隙 |
| 当前收益 | 当前被看见的改变 |
| 当前代价 | 风险、消耗、暴露、误判 |
| 下一阈值 | 进入下一阶段还缺什么 |
