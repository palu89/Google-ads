# 卷章结构

## 推荐卷结构

| 卷名 | 时间窗 | 核心任务 | 变量阶段 |
| --- | --- | --- | --- |
| 卷一《断裂地带》 | 1927-1928 | 在清洗与断裂中活下来，并保住第一条线 | 注入 |
| 卷二《山线》 | 1928-1931 | 把零散求生变成可重复的组织能力 | 扎根 |
| 卷三《转移》 | 1931-1936 | 在围剿、外压与路线调整中保住积累 | 扩散 / 对抗 |
| 卷四《共敌与暗流》 | 1936-1940 | 在统一战线框架中处理合作与猜忌 | 对抗 |
| 卷五《边区试验》 | 1940-1945 | 长期消耗、治理与干部留存 | 放大 |
| 卷六《复燃》 | 1945-1949 | 接收、接管、决战与国家重构 | 放大 / 改写 |
| 卷七《新国与外战》 | 1949-1953 | 新政权巩固、外部战争与终局校验 | 改写 |

## 样板卷建议

- `[高可信推断]` 样板卷应优先写 `卷一《断裂地带》`。
- 理由：
  - 最能体现严肃历史模拟的代价。
  - 最能把变量写成“从人和线开始”，而不是“从胜负开始”。
  - 最能测试项目的虚构边界有没有守住。

## 样板卷最低交付

1. 一句梗概。
2. 主角功能位。
3. 注入事件。
4. 三个连续后果。
5. 一份变量台账。
6. 五到八章细纲。
