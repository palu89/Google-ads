# 虚构边界

## 总原则

- 先有史实基线，再有架空偏离。
- 架空不是任意改写，而是从可追踪的局部扰动出发。
- 真实历史人物可以被影响，但不能被轻易“重写人格”。

## 可直接虚构的部分

- 县、区、乡、团、营、连级以下的虚构人物与关系网。
- 未被史料详细记录的小规模联络、转移、护送、救治、征粮、情报交接。
- 私人动机、日常摩擦、组织内部微观情绪。
- 地方层面的隐秘行动，只要不直接撞毁一级锚点。

## 需要严格克制的部分

- 真实人物的公开决策与公开行踪。
- 重大会议、战役、政策、建制名称、口径与时代术语。
- 武器、交通、通信、医疗、粮秣条件。
- 不同年份对外援助、装备来源与部队成分。

## 改写阈值

- `注入阶段`：只允许影响局部个人命运、一次任务成败、小规模联络质量。
- `扎根阶段`：允许影响一个点位、一个班排、一个交通站、一条补给线。
- `扩散阶段`：允许影响一个县、一个游击区、一个局部组织网络。
- `对抗阶段`：允许与既有制度、路线、派系、敌对力量发生反作用。
- `放大阶段`：允许影响战役准备、干部留存率、政策执行节奏。
- `改写阶段`：只有在前五阶段证据链完整时，才允许触碰全国性结局。

## 真实人物处理规则

- 初期以“隔着组织层级被影响”为主，不宜直接贴身改写。
- 不允许真实人物在缺乏充分动机时无条件信任主角。
- 不允许用后见之明把真实人物写成“等主角来拯救”的空壳。

## 禁止的写法

- 一次见面就改变中央路线。
- 一次战斗就改写全国政治格局。
- 凭现代百科知识直接解决年代性的工业、财政、组织难题。
- 把复杂历史阵营写成单一善恶结构。
