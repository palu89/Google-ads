# Google Ads Appeal

Use this skill when the user needs appeal strategy, not just generic compliance advice.

## Use this for
- account suspension appeals
- ad rejection appeals
- remediation sequencing before appeal submission
- evidence package design for reviewer re-check

## Workflow
1. Identify the exact policy or suspension class.
2. Separate reversible issues from structural blockers.
3. Produce the remediation order before drafting text.
4. Build the appeal package: facts, fixes, evidence, and reviewer guidance.
5. If a final letter or comments box draft is needed, hand off to `googleads-appealtxt`.

## Output format
Policy judgment -> remediation plan -> appeal package checklist -> escalation risks -> evidence map
