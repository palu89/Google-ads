# Google Ads Appeal Text

Use this skill only after the factual remediation state is clear.

## Use this for
- Google Ads comments box drafts
- reviewer-facing remediation summaries
- English appeal letters
- short, high-credibility resubmission text

## Writing rules
1. State the problem, the fix, and the verification evidence plainly.
2. Do not over-argue or speculate.
3. Avoid emotional language, blame, or unsupported claims.
4. Keep it reviewer-friendly and easy to verify.
5. If licensing or regulator evidence is involved, cite it cleanly.

## Output format
Final appeal text -> short variant -> evidence checklist to attach
