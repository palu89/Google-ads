# India Financial Verification Scope

Use this note when you need the practical boundary for India financial-services verification.

## Core rule
- Google states that financial-services advertisers must be verified to show financial services ads in India.
- The same verification requirement can also apply when ads are shown to Indian users who appear to be seeking financial services.

## Operating boundary
- Treat India financial verification as a strict policy gate, not a vague best-practice suggestion.
- Do not describe it internally as if every India-targeted account automatically follows the exact same workflow.
- Use the service type, user intent, and policy classification together when deciding whether verification is relevant.

## Practical use
- Use this file in audits, sales explanations, and account triage when the discussion is specifically about India scope.
- Pair it with `finance_compliance.md` for the broader finance policy context and `policies.md` for the general policy hub.
